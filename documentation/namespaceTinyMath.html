BoundingBoxAware *  collided_with,
VerticalCollisionDirection  colliding_object_direction 
)
inline

This method is used to compute collision details from a collision that happens between 2 BoundingBoxAware objects.

This method computes collision after checking for a potential collision first, so it is safe to call even if there is no potential collision between 2 objects.

Parameters
colliding_objectA pointer to the colliding object that maybe in a potential conclusion.
collided_withA pointer to the second object that maybe in a potential conclusion with colliding_object.
colliding_object_directionThe vertical direction of the movement of colliding_object.
Returns
A Collision object containing details of the collision.
