◆ GetImageResourceBMP()

SDL_Texture * ResourceManager::GetImageResourceBMP ( const std::string &  resource_id,
SDL_Renderer *  renderer 
)

This method is used to retrieve the image resources available in the game. The resources are loaded in a lazy fashion.

Since only a pointer to the image resource is passed, it is ensured that requesting the same resource multiple times will not flood the memory with redundant copies. This method is optimized for BMP files.

Parameters
resource_idThe unique resource ID for the asset. This ID is mostly the location of the corresponding file from which the resource is loaded.
rendererA pointer to the SDL_Renderer* that is used to create texture for the image resource (if the resource is not loaded into memory yet.
Returns
A pointer to the SDL_Texture of the rendered image resource as a SDL_Texture.

◆ Shutdown()

void ResourceManager::Shutdown ( )

This method is used to shutdown the resource manner in a proper way and should be called whenever the user exits the game.

This method clears all the mapping of the loaded resources and deletes all dynamically allocated memory. Calling this method will nullify the current instance of GameResourceManager.
